
# Github:

前端: [https://github.com/czero1995/fancy-store](https://github.com/czero1995/fancy-store)

### 安装依赖

    # 预渲染需要安装PrerenderSPAPlugin库，npm install会无法安装Chromuim，建议使用cnpm
    npm install -g cnpm --registry=https://registry.npm.taobao.org
    cnpm install

### 启动服务

    npm run serve

### 构建生产环境

    npm run build
